    Operating Systems Terminology

    Term DEFINITION
    Operating System An operating system is the software that controls the overall operation of a computer. It provides the means by which a user can store and retrieve files, provides the interface by which a user can request the execution of program, and provides the environment necessary to execute the programs requested.
    Job Queue It is a storage organization in which objects, like programming, data and directions are ordered in first-in, first-out fashion.
    FIFO It stands for first-in first-out.
    Interactive processing An operating system that allowed a program being executed to carry on a dialogue with the user through remote terminals.
    Time-sharing Operating systems that provided service to multiple users at the same time.
    Multiprogramming An operating system technique in which time is divided into intervals and then the execution of each job is restricted to only one interval at a time. At the end of each interval, the current job is temporarily set aside and another is allowed to execute during the next interval.
    Application software Application software consists of the programs for performing tasks particular to the machine's utilization.
    System Software It performs those tasks that are common to computer systems in general. In a sense, the system software provides the infrastructure that the application software requires, in much the same manner as a nation’s infrastructure (government, roads, utilities, financial institutions, etc.) provides the foundation on which its citizens rely for their individual lifestyles.
    Utility Software It consists of programs for performing activities that are fundamental to computer installations. utility software include software to compress and decompress data, software for playing multimedia content, and software for handling network communication.
    Graphical User Interface (GUI) The system perform tasks in which objects to be manipulated, such as files and programs, are represented pictorially on the display as icons. These systems allow users to issue commands by using one of several common input devices.
    Shell Shells are older interface communicated with users through textual messages using a keyboard and monitor screen
    Kernel The internal part of an operating system is called the kernel
    Device driver Are the software units that communicate with the controllers to carry out operations on the peripheral devices attached to the machine
    File manager It maintains records of all the files stored in mass storage
    Memory manager It is charged with the task of coordinating the machine’s use of main memory
    Virtual memory A large fictional memory by paging is called virtual memory.
    Boot Strapping The procedure that transfers the operating system from mass storage (where it is permanently stored) into main memory (which is essentially empty when the machine is first turned on).
    ROM It contains a collection of software routines for performing fundamental input/output activities such as receiving information from the keyboard, displaying messages on the computer screen, and reading data from mass storage.
    Deadlock A situation where two programs are trying to access the same resources and are effectively preventing each other from accessing the resource.
    Spooling It is a form multi programming for the purpose of copying data between different devices
    Super user(or administrator) Those are highly privellged accounts used by specialized IT employees.
    Multi-factor authentication It is a login process in which the user is required additional information inluding the password.
