    Networking and the Internet

    Network Classifications Terms Definitions
    LAN LAN stands for "Local Area Network." It is a network of interconnected computers and devices that are located in a relatively small geographic area.
    MAN A MAN, or Metropolitan Area Network, is a network that spans a larger geographic area than a Local Area Network (LAN).
    WAN A WAN, or Wide Area Network, is a network that spans a large geographic area, typically a country, continent, or even the entire world.
    Network Models Definitions
    bus network In a bus network, data is transmitted down the cable, and each device on the network receives the data simultaneously.
    star network In a star network, each device has a dedicated point-to-point connection to the central hub, which acts as a central point of control and distribution for data traffic.
    Protocols Definitions
    CSMA/CD Ethernet CSMA/CD stands for "Carrier Sense Multiple Access with Collision Detection." It is a set of rules and a network access method that was used in early Ethernet networks to manage how devices share the network medium and avoid data collisions.
    CSMA/CA Wifi CSMA/CA stands for "Carrier Sense Multiple Access with Collision Avoidance." It is a network access method and protocol used in Wi-Fi (Wireless Fidelity) networks to manage how devices share the wireless communication medium and avoid data collisions.
    Combining Networks Definitions
    repeater A repeater is a network device that is used to extend the range and improve the signal quality of data transmissions within a network.
    bridge A bridge is a network device or component that operates at the data link layer (Layer 2) of the OSI model and is used to connect and manage traffic between two or more network segments, effectively creating a single, larger network.
    switch A switch is a network device that operates at the data link layer (Layer 2) of the OSI model. It is designed to connect devices within a local area network (LAN) and efficiently manage the traffic.
    router A router is a network device that operates at the network layer (Layer 3) of the OSI model and is responsible for forwarding data packets between different networks.
    gateway A gateway, in the context of computer networks, is a device or software program that serves as a bridge between different networks or network protocols.
    Client/server model Definitions
    P2P: peer-to-peer Peer-to-Peer (P2P) refers to a decentralized network architecture where participants, often referred to as "peers," interact directly with one another, sharing resources, services, or data without the need for a centralized server or authority.
    distributed system It is a network of interconnected computers and devices that work together as a single, unified system
    cloud computing Cloud computing refers to the delivery of various computing services, including servers, storage, databases, networking, software, analytics, and more, over the internet.
    Internet Addressing Definitions
    Tier 1 Tier 1 network is a classification used to describe the highest level or top-tier networks in the global internet infrastructure hierarchy.
    Tier 2 Tier 2 networks are regional or national ISPs that provide internet connectivity to a specific geographic area or region.
    Tier 3 The term "Tier 3" in the context of networks and internet service providers (ISPs) is not a standard industry classification like Tier 1 or Tier 2. Instead, it is sometimes informally used to describe smaller, local, or more specialized ISPs that operate on a smaller scale.
    IP address Definitions
    ICANN ICANN, or the Internet Corporation for Assigned Names and Numbers, is a nonprofit organization responsible for coordinating and managing several key elements of the global internet's infrastructure.
    DNS DNS serves as the "phone book" of the internet, helping users access websites, send emails, and connect to various online services by translating human-readable domain names into IP addresses.
    TLD: Top-level domain A top-level domain (TLD) is the highest level of the domain name system (DNS) hierarchy and is the last segment of a domain name in a web address.
    Country-code TLDs Country code top-level domains (ccTLDs) are specific categories of top-level domains (TLDs) within the Domain Name System (DNS) that are associated with individual countries, territories, or regions.
    Internet Protocols Definitions
    FTP FTP, or File Transfer Protocol, is a standard network protocol used for transferring files between a client and a server on a computer network, typically the internet.
    HTTP HTTP, or Hypertext Transfer Protocol, is an application layer protocol used for transmitting hypermedia documents, such as web pages and resources, over the World Wide Web.
    SMTP SMTP, or Simple Mail Transfer Protocol, is an internet protocol used for sending, relaying, and delivering email messages between email servers.
    World Wide Web Definitions
    URL URL, or Uniform Resource Locator, is a standardized address used to identify and locate resources on the internet.
    Client-side activities Client-side activities, in the context of web development and internet technologies, refer to the processes and operations that occur on the user's device (the client) when interacting with a website or web application.
    Server-side activites Server-side activities, in the context of web development and internet technologies, refer to the processes and operations that occur on the web server in response to a user's request when interacting with a website or web application.
    Internet Four-Layer Software Definitions
    Application layer The application layer is responsible for managing communication between software applications and end-users, making it the layer closest to the end-users and the interface through which users interact with network services and resources.
    Transport layer The transport layer is responsible for the end-to-end communication and data transport between devices on a network.
    Network layer The network layer is responsible for the routing of data packets between devices on different networks, and it plays a crucial role in enabling internetwork communication.
    Link layer It is responsible for the reliable transmission of data frames between directly connected devices over a physical medium, such as a local area network (LAN).
    Internet is: Definitions
    Packet-switching network A packet-switching network is a type of data communication network that transmits data in the form of discrete packets or frames.
    Fault tolerant Fault tolerance is a characteristic of a system or technology that enables it to continue operating and providing its intended services and functions, even in the presence of hardware failures, software errors, or other unexpected issues.
    Redundant "Redundant" is a term used to describe the existence of excess or duplicate components, systems, or resources within a system or process.
    Cybersecurity Definitions
    Malware Malware, a portmanteau of "malicious software," refers to any type of software or code specifically designed to infiltrate, damage, disrupt, or gain unauthorized access to computer systems, networks, or devices, with the intent to cause harm or exploit the affected system for various purposes.
    Virus A computer virus is a type of malicious software (malware) that is designed to replicate and spread from one computer to another by attaching itself to legitimate programs or files.
    Worm A computer worm is a type of self-replicating malware that spreads across computer networks and systems by exploiting vulnerabilities and weaknesses in software, hardware, or network protocols. Unlike computer viruses, which attach themselves to existing files or programs, worms are standalone programs that do not require a host file for propagation.
    Trojan Horse A Trojan horse, often simply referred to as a "Trojan," is a type of malicious software (malware) that disguises itself as a legitimate and harmless program or file to deceive users into executing it.
    Spyware Spyware is a type of malicious software (malware) that is designed to secretly collect information from a user's computer or device without their knowledge or consent.
    Phishing Phishing is a form of cyberattack and online fraud in which attackers, posing as trustworthy entities, attempt to deceive individuals into providing sensitive information, such as login credentials, financial details, personal information, or credit card numbers.
    DDoS attack A Distributed Denial of Service (DDoS) attack is a malicious cyberattack in which a large number of compromised or controlled computers, known as "bots" or "zombies," are used to flood a target system or network with a massive volume of traffic or requests.
    Protection Definitions
    Firewall A firewall is a network security device or software application that is designed to monitor and control incoming and outgoing network traffic based on predetermined security rules.
    Spam filter A spam filter is a software or hardware mechanism designed to identify and filter out unsolicited and unwanted email messages, often referred to as "spam."
    Proxy server A proxy server, often simply referred to as a "proxy," is an intermediary server or software application that acts as a gateway between a client device (such as a computer or smartphone) and another server or service on the internet.
    Antivirus software Antivirus software, often referred to as antivirus or anti-malware software, is a type of computer program or application designed to detect, prevent, and remove malicious software, including viruses, malware, worms, Trojans, spyware, adware, and other threats that can infect and compromise computer systems and data.
    Cryptography Cryptography is the science and practice of securing communication and data by converting it into a code or cipher that can only be deciphered by individuals who possess the necessary decryption key or information.
    Public key encryption Public key encryption, also known as asymmetric encryption, is a cryptographic method that uses a pair of keys to secure the exchange of data between two parties.
